# Code & Review 工作流

这是一个自动化的"写代码 + Review"工作流，适用于任何项目。

## 使用方法

在 Claude Code 中输入：
```
/workflow code-and-review "实现一个 XXX 功能"
```

## 工作流程

1. **规划阶段** (Plan Agent)
   - 分析需求
   - 设计实现方案
   - 识别需要修改的文件

2. **实现阶段** (Sonnet Agent - 快速实现)
   - 根据规划写代码
   - 运行基础测试
   - 生成实现报告

3. **Review 阶段** (Opus Agent - 深度审查)
   - 代码质量检查
   - 架构合理性评估
   - 性能和安全审查
   - 提出改进建议

4. **优化阶段** (Sonnet Agent - 快速迭代)
   - 根据 review 意见修复问题
   - 再次测试

5. **总结阶段**
   - 生成变更摘要
   - 列出关键文件和改动
   - 建议后续工作

## 优势

- ✅ 无需手动切换模型
- ✅ 自动化整个流程
- ✅ Sonnet 快速写代码，Opus 深度审查
- ✅ 完整的审查记录
- ✅ 适用于任何编程语言/项目
